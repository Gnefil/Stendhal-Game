# Stendhal
---
Stendhal is a MMORPG based on Marauroa game engine. It is an open source game which full detail can be found [here](https://github.com/arianne/stendhal).

This is **modified** version that runs on localhost. [Instructions](https://stendhalgame.org/wiki/Stendhal_on_Eclipse) to run the game in Eclipse.

Modifications include:
- 7 functional bug fix
- 3 feature implementation
- 2 releases
